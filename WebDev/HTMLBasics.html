  Below is a list of text formatting elements, in tabular form:

HTML Text Formatting Elements

Tag Description
<b> Defines bold text
<strong> Defines important text
<em> Defines emphasized text 
<i> Defines italic text
<small> Defines smaller text
<sub> Defines subscripted text
<sup> Defines superscripted text
<ins> Defines inserted text
<del> Defines deleted text
<mark> Defines marked/highlighted text
<q> Defines a quoted text
<blockquote> This is generally used to quote a large piece of text.
It tab spaces the quote, like this. Note that tabs and spaces in the html file, in this context, won't be visible in the web document. However, if <blockquote> is used in a <pre> tag context, for example, the tabs and spaces will be present.
<abbr> Defines an abbreviation, e.g. lol. Has no effect on web document, but provides useful information to browsers, translation systems and search-engines.
<bdo> Bi-Direction Override. Used to override text direction. E.g. right to left
